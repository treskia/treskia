# Treskia

Treskia is a personal-finance iOS app for India. It reads the transaction-alert emails your banks already send to Gmail and turns them into a spending view — parsing and AI inference happen entirely on the phone, and no email or transaction data reaches any server.

This repo is the app's public site (landing page, privacy policy, terms of use), served with GitHub Pages at [treskia.com](https://treskia.com/). Design source and mockups live in [`design/`](design/).
